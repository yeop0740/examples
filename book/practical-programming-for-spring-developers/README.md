## Practical Programming For Spring Developers

### 14. 테스트 대역

#### 14.2 Stub

stub은 미리 준비된 값을 반환하는 대역 객체
외부 API를 호출하거나 I/O 등을 수반하는 데이터 관련 연산 작업을 대체

stub을 사용하여 실제로 다른 시스템을 호출하지 않고 실패/예외 케이스를 테스트할 수 있다.
-> 항상 같은 상황을 만들어준다. -> 테스트가 결정적으로 동작할 수 있도록 도와준다.

> Dummy, Stub을 만든 것처럼 직접 대역 객체를 만들 수도 있고, Mockito 와 같은 프레임워크를 사용할 수도 있다.

stub은 메서드 스텁이라고도 불린다. 어떤 객체의 메서드의 행동을 다르게 유도할 때에 사용되기 때문이다.
